# ToDo List

Simple Todo-List
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.
```bash
git clone the Repo
```

## Usage

```python

cp .env.example .env
docker-compose up -d
docker exec -it todolist-app bash
run composer install 
php artisan migrate

```
